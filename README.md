## 設計思想

Udonは非常に制限されたC#ランタイムです。
厳しい環境で最高のユーザー体験を提供するために同期関数を動的に生成してハードコーディングしています。
これにより安定していて高速なユーザー間でのリアルタイム同期処理が可能になっています。
